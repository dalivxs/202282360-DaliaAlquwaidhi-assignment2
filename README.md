# Assignment 1 – Personal Portfolio (Foundation & AI Integration)

## Project Description
This is a simple, responsive personal portfolio website built using HTML, CSS, and JavaScript. It includes three main sections (About, Projects, Contact) and basic interactivity such as a theme toggle and form interaction.

## Features
- About Me section with a short introduction and tagline
- Projects section with two project cards (title, description, placeholder image)
- Contact section with a form (front-end only, no backend)
- Responsive design for desktop, tablet, and mobile
- JavaScript interactivity (theme toggle + basic form behavior)

## Folder Structure
- `index.html` – main page
- `css/styles.css` – styling + responsive layout
- `js/script.js` – JavaScript interactivity
- `assets/images/` – images (placeholders)
- `docs/ai-usage-report.md` – AI tool usage report
- `docs/technical-documentation.md` – technical documentation

## How to Run Locally
1. Clone the repository:
   - `git clone <YOUR_REPO_LINK>`
2. Open the folder in VS Code.
3. Open `index.html` in a browser (right-click → Reveal in File Explorer → double-click `index.html`).
